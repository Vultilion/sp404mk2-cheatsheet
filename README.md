# SP-404MK2 Web Cheat Sheet

An interactive, web-based workflow cheat sheet for the Roland SP-404MK2. It provides quick access to button combinations, shortcuts, and various workflows.

## 🔗 Live Version
Check out the live version here: **[SP-404MK2 Cheat Sheet](https://vultilion.github.io/sp404mk2-cheatsheet/)**

## ✨ Features
* **Interactive Search**: Instantly find specific workflows, effects, or system settings.
* **Bilingual Support**: Toggle between English and German.
* **Print-Optimized**: Designed to export cleanly as a PDF or physical print.
* **Responsive Design**: Works perfectly on mobile and desktop.

## 🚀 Usage / Setup
This is a simple, vanilla HTML/CSS/JS application. No build step is required! 
Just clone the repository and open `index.html` in your browser.
